# Separator Images

Decorative line images that appear above ad sections.

## Dimensions

- **Width**: Full width of ad column (2.25in / 162px at 72dpi)
- **Height**: 0.5in / 36px at 72dpi
- **Format**: PNG, JPG, or WebP with transparent or white background

## Notes

- Images are scaled to fit width, cropped to fixed height
- Use subtle, decorative line designs
- White/transparent backgrounds work best with the white ad frames
